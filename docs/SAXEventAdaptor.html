Class: SAXEventAdaptor

SAXEventAdaptor(delegate)

EventAdaptor for SAX parser. The following events are piped through the stream as tokens
  • attribute - SAXEventAdaptor~AttributeToken
  • cdata - SAXEventAdaptor~CDataToken
  • closecdata - SAXEventAdaptor~CloseCDataToken
  • closenamespace - SAXEventAdaptor~CloseNamespaceToken
  • closetag - SAXEventAdaptor~CloseTagToken
  • comment - SAXEventAdaptor~CommentToken
  • doctype - SAXEventAdaptor~DoctypeToken
  • opencdata - SAXEventAdaptor~OpenCDataToken
  • opentag - SAXEventAdaptor~OpenTagToken
  • opentagstart - SAXEventAdaptor~OpenTagStartToken
  • opennamespace - SAXEventAdaptor~OpenNamespaceToken
  • processinginstruction - SAXEventAdaptor~ProcessingInstructionToken
  • script - SAXEventAdaptor~ScriptToken
  • sgmldeclaration - SAXEventAdaptor~SGMLDeclarationToken
  • text - SAXEventAdaptor~TextToken
