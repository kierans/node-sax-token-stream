    EventAdaptor for SAX parser.

    The following events are piped through the stream as tokens

    • <b>attribute</b> - SAXEventAdaptor~AttributeToken
    • <b>cdata</b> - SAXEventAdaptor~CDataToken
    • <b>closecdata</b> - SAXEventAdaptor~CloseCDataToken
    • <b>closenamespace</b> - SAXEventAdaptor~CloseNamespaceToken
    • <b>closetag</b> - SAXEventAdaptor~CloseTagToken
    • <b>comment</b> - SAXEventAdaptor~CommentToken
    • <b>doctype</b> - SAXEventAdaptor~DoctypeToken
    • <b>opencdata</b> - SAXEventAdaptor~OpenCDataToken
    • <b>opentag</b> - SAXEventAdaptor~OpenTagToken
    • <b>opentagstart</b> - SAXEventAdaptor~OpenTagStartToken
    • <b>opennamespace</b> - SAXEventAdaptor~OpenNamespaceToken
    • <b>processinginstruction</b> - SAXEventAdaptor~ProcessingInstructionToken
    • <b>script</b> - SAXEventAdaptor~ScriptToken
    • <b>sgmldeclaration</b> - SAXEventAdaptor~SGMLDeclarationToken
    • <b>text</b> - SAXEventAdaptor~TextToken
